# Safaricom M-Pesa SDK

This is an opinionated fluent Safaricom M-Pesa SDK
## Tech Stack

**Server:** Nginx, Apache, PHP

  
## Installation 

Clone the project using the following

```bash 
  https://github.com/Rndwiga/mpesa-sdk.git
  cd mpesa-sdk
  composer update
```
